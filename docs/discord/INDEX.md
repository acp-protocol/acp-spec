# Discord Server Documentation Index

This index provides quick access to all Discord server documentation and resources.

## Main Documentation

### [Discord Server README](./README.md)
Complete overview of the Discord server, including channel structure, permissions, resources, and community guidelines.

### [Onboarding Guide](./onboarding.md)
Step-by-step guide for new members, including welcome flow, role selection, getting started steps, and educational content series.

### [Bot Commands Reference](./bot-commands.md)
Complete reference for all Discord bot commands, including information commands, utility commands, and community commands.

### [Moderation Guide](./moderation-guide.md)
Comprehensive moderation structure, roles, guidelines, conflict resolution, and community management processes.

### [Content Calendar](./content-calendar.md)
Weekly, monthly, and quarterly content planning, including regular posts, events, educational series, and engagement activities.

### [Bot Automation Guide](./bot-automation.md)
Configuration for Discord MCP server automation, including welcome system, GitHub integration, help channel management, and automation scripts.

### [Welcome Messages](./welcome-messages.md)
Templates for welcome messages, DMs, getting started messages, rules, role selection, and progress tracking.

### [Launch Checklist](./launch-checklist.md)
Complete pre-launch checklist, testing procedures, launch day activities, and success criteria.

### [Growth Strategy](./growth-strategy.md)
Growth goals, launch activities, community outreach, content marketing, engagement tactics, and metrics tracking.

## Quick Reference

### For New Members

1. Start with [Onboarding Guide](./onboarding.md)
2. Read [Discord Server README](./README.md)
3. Check [Bot Commands Reference](./bot-commands.md) for available commands
4. Review [Welcome Messages](./welcome-messages.md) for server information

### For Moderators

1. Read [Moderation Guide](./moderation-guide.md)
2. Review [Bot Automation Guide](./bot-automation.md)
3. Check [Content Calendar](./content-calendar.md) for scheduled content
4. Use [Welcome Messages](./welcome-messages.md) templates

### For Maintainers

1. Review [Launch Checklist](./launch-checklist.md) before launch
2. Plan with [Growth Strategy](./growth-strategy.md)
3. Configure with [Bot Automation Guide](./bot-automation.md)
4. Monitor with [Content Calendar](./content-calendar.md)

### For Contributors

1. Read [Discord Server README](./README.md) for community guidelines
2. Check [Bot Commands Reference](./bot-commands.md) for helpful commands
3. Review [Moderation Guide](./moderation-guide.md) for contribution guidelines
4. Share work using showcase guidelines
